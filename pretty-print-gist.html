Pretty Print Gist Bookmarklet

The following link can be dragged to the bookmarks toolbar and can be used as a `Bookmarklet` so that whenever you have a gist that you might want to print or save as a pdf file or any other format. it can be viewed without distractions.

Installation

  • Drag the link to your bookmark toolbar
  • Visit any gist that you want to view without distractions
  • Click the newly installed bookmark
